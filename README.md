## Como ejecutar

Descarga la carpeta o clona el repo, y abre una terminal ahi y luego ejecuta:
```
python -m pip install --upgrade virtualenv
python -m virtualenv env
source env/bin/activate
pip install pyqt6 pyqt6-webengine
python main.py
```
